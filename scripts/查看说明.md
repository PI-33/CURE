方式二：inspect 工具（按需查看当前数据）
训练过程中或训练结束后，在项目根目录运行：

# 总览所有问题
python scripts/inspect_generations.py
# 看第3题的详情（所有代码和测试的摘要）
python scripts/inspect_generations.py -p 3
# 看第3题的第0个代码完整内容 + 执行结果
python scripts/inspect_generations.py -p 3 -c 0
# 看第3题的第2个测试完整内容 + 通过情况
python scripts/inspect_generations.py -p 3 -t 2
# 指定历史文件（如果你手动备份了某步的outputs-rl文件）
python scripts/inspect_generations.py --file path/to/outputs-rl-xxx.json -p 0
